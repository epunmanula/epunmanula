EmmE Video Speed Master is a powerful and lightweight browser extension designed to give you absolute control over HTML5 video playback speeds. Fine-tune your viewing experience beyond fixed options.

### Key Features

* **Flexible Speed Control:** Adjust video speed anywhere from `0.1x` up to `16.0x`.
* **Global Hotkeys:** Control speed without ever leaving full screen. Use keys to increase/decrease, seek, and reset.
* **Settings Sync:** Save and sync your playback preferences across multiple devices.
* **Universal Compatibility:** Works seamlessly across major platforms like YouTube, Netflix, Coursera, Udemy, and almost any site with an HTML5 video player.

### Installation & Usage

Directly from the extension panel or use keyboard shortcuts:

| Action | Shortcut Key | Description |
| :--- | :---: | :--- |
| **Decrease Speed** | `[` | Speeds down the video by 0.1x increments. |
| **Increase Speed** | `]` | Speeds up the video by 0.1x increments. |
| **Reset Speed (1.0x)**| `R` | Instantly resets the video to normal speed. |
| **Toggle UI Console** | `H` | Shows or hides the EmmE control overlay. |
| **Seek Backwards** | `Left Arrow (←)` | Jump back in the video timeline. |
| **Seek Forwards** | `Right Arrow (→)` | Jump forward in the video timeline. |
